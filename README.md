# 🏠 서울시 부동산 실거래가 데이터 분석

## 📌 프로젝트 개요
서울시 공공데이터 포털에서 제공하는 부동산 실거래가 데이터를 활용하여
2018년~2024년 서울시 자치구별, 건물용도별 가격 트렌드와 투자 수익률을 분석한 프로젝트입니다.

## 🗓 진행 기간
2026.01.26 ~ 2026.02.03

## 🛠 사용 기술
`SQL` `Python` `Pandas` `NumPy` `Matplotlib` `Seaborn`

## 📂 데이터 소개
- **원본 데이터**: 833,614건 / 21개 컬럼
- **전처리 후**: 약 777,000건 / 15개 컬럼
- **분석 대상 기간**: 2018년 ~ 2024년
- **건물 용도**: 아파트, 연립다세대, 단독다가구, 오피스텔

## 📋 데이터 주요 컬럼 정보
| 컬럼명 | 설명 |
|--------|------|
| CGG_NM | 자치구명 |
| STDG_NM | 법정동명 |
| BLDG_NM | 건물명 |
| CTRT_DAY | 계약일 |
| THING_AMT | 물건금액(만원) |
| ARCH_AREA | 건물면적(㎡) |
| ARCH_YR | 건축년도 |
| BLDG_USG | 건물용도 (아파트/연립다세대/단독다가구/오피스텔) |
| AMT_PER_PYEONG | 평당가(만원/평) |
| QUARTER | 분기 (예: 2018Q1) |

## 🔍 분석 내용

### 1. 데이터 전처리 (Preprocessing)
- 컬럼명 한글 → 영문 변환 및 불필요 컬럼 제거
- 2024년 4분기(10월까지만 집계) 제거 — 분기 분석 정확도 확보
- 건물금액 상위 1.5% 초고가 매물 제거
- 취소 거래, 자치구명 오류 21건, 법정동명 오류 3건 제거
- 건축년도 결측치 → 9999 처리 (데이터 가용성 극대화)
- 오피스텔 이상치 4건 실거래 사이트 직접 검증 → 1건만 제거 (트윈시티 남산)

### 2. 분석 및 시각화 (Analysis)
- 건물용도별 연도/분기별 평균 거래금액 및 평당가 추이
- 자치구별 **QoQ(전분기 대비) 증감률 히트맵** — 누적 수익률 기준 정렬
- 특정 지역·건물 **투자 수익률 시뮬레이션** (CAGR 계산)
- 자치구별 건물용도별 분기별 평당가 추이 (서울 전체 평균과 비교)
- 금천구 오피스텔 **거래량 상위 5개 건물 평당가 추이** 심층 분석

## 📊 주요 분석 결과
### 📍 고객 A — 종로구 아파트 투자 실패 케이스

> 2022년 1월 종로구 아파트 10억 투자 → 22년 3~4분기 최대 낙폭 **-34.42%** → 공포 매도 → **약 3억 3천만원 손실 확정**
> 
- 종로구는 원래 거래량이 적어 외부 충격에 취약한 시장
- 하락 후 불과 1개 분기 만에 강력 반등 → 조금만 더 기다렸다면 수익 전환 가능
- **교훈**: 거래량이 충분히 뒷받침되는 시장 선택 + 시장 신호를 읽는 인내심 필요

### 📍 고객 B — 금천구 오피스텔 투자 전략 검증

> "평당가 하위권 지역 중 누적수익률 상위 = 가성비 투자" 전략 → 백테스팅 결과 **3억 투자 시 2억 5천만원 수익 예상**
> 
- 그러나 실제 금천구 오피스텔 개별 매물 시계열 분석 결과 → **신축 고가 매물 신규 거래로 인한 통계적 착시**
- 기존 구축 오피스텔은 신축 평당가를 따라가지 못하고 정체
- **교훈**: 자치구 평균 수익률만 볼 것이 아니라 **개별 매물 시계열 검증** 필수

## ⚠️ 한계점 & 개선사항
