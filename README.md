# 🏠 서울시 부동산 실거래가 데이터 분석

## 📌 프로젝트 개요
서울시 공공데이터 포털에서 제공하는 부동산 실거래가 데이터를 활용하여
2018년~2024년 서울시 자치구별, 건물용도별 가격 트렌드와 투자 수익률을 분석한 프로젝트입니다.

## 🗓 진행 기간
2026.01.26 ~ 2026.02.03

## 🛠 사용 기술
`SQL` `Python` `Pandas` `NumPy` `Matplotlib` `Seaborn`

## 📂 데이터 출처
- 서울 열린데이터 광장 - 서울시 부동산 실거래가 정보
- 원본 데이터: 833,614건 / 전처리 후 분석 데이터: 약 777,000건

## 📋 데이터 컬럼 정보
| 컬럼명 | 설명 |
|--------|------|
| CGG_NM | 자치구명 |
| STDG_NM | 법정동명 |
| BLDG_NM | 건물명 |
| CTRT_DAY | 계약일 |
| THING_AMT | 물건금액(만원) |
| ARCH_AREA | 건물면적(㎡) |
| ARCH_YR | 건축년도 |
| BLDG_USG | 건물용도 (아파트/연립다세대/단독다가구/오피스텔) |
| AMT_PER_PYEONG | 평당가(만원/평) |
| QUARTER | 분기 (예: 2018Q1) |

## 🔍 분석 내용

### 1. 데이터 전처리 (Preprocessing)
- 컬럼명 한글 → 영문 변환 및 불필요 컬럼 제거
- 분석 기간 필터링: 2018년 ~ 2024년 (820,169건)
- 취소 거래 제거, 결측치 처리
- 파생 변수 생성: 계약연도, 분기, 건물연령, 평당가(AMT_PER_PYEONG), 면적당 금액(AMT_PER_AREA)
- 건물용도별 이상치 탐지 및 처리 (아파트/오피스텔 스캐터플롯 활용)

### 2. 분석 (Analysis)
- **자치구별 QoQ(전분기 대비) 증감률 히트맵**: 누적 수익률 기준 정렬, 서울시 전체 평균과 비교
- **수익률 시뮬레이션**: 특정 지역·건물의 2018년 매수 → 2024년 매도 시 CAGR 계산
- **자치구별 건물용도별 분기별 평당가 추이 시각화**: 서울 평균과 비교하는 라인 차트
- **금천구 오피스텔 심층 분석**: 거래량 상위 5개 건물의 분기별 평당가 추이

## 📊 주요 분석 결과
- 아파트 기준 누적 수익률 1위 자치구: **금천구 (115.95%)**
- 서울시 전체 아파트 누적 수익률: **54.38%**
- 서대문구 대현동 아파트 2018→2024 수익률: **49.12% (CAGR 6.89%)**
- 오피스텔 이상치: 롯데월드타워(송파구), 피엔폴루스(강남구 청담동) 등 고가 실거래 확인 후 유지

## 💡 느낀 점 & 배운 점
- 대용량 데이터(80만건+) 처리 시 메모리 효율화의 중요성을 경험
- 이상치 판단 시 단순 통계값이 아닌 실제 거래 데이터를 직접 검증하는 과정의 중요성 학습
- 분기(QUARTER) 파생변수를 활용한 시계열 분석 및 피벗 테이블 활용 능력 향상
